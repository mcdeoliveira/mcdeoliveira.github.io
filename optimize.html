Optimize API

The Optimize API gives you access to a cloud-based optimizer that can be used to solve common optimization problems in your app or online application.

Problems can be formulated and solutions can be calculated and retrieved via an AJAX call, enabling easy integration into your Python, Javascript, Java, or any other modern application language you use.
